# Hori Fujimoto Lab LaTeX Template

このテンプレートはpLaTeX環境を想定して作成してあります。
各テンプレートのpreambleは最小限の構成になっているので足りない場合は適宜追加してください。

## フォルダの説明

## eeis
電気系輪講のテンプレート

## faq
LaTeXに関する説明およびFAQ

## happyou
発表会資料のテンプレート

## houkoku
報告会資料のテンプレート

## ieee
IEEEのテンプレート

## ieej
電気学会のテンプレート

## preamble
プリアンブルの.styファイルのサンプル

自分のプリアンブルが洗練されてきたら
```
preamble_name.sty
```
というファイル名で是非追加してください。

## thesis
